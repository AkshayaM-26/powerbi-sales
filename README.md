# 📊 Sales Performance Dashboard 
## Project Overview
This project presents an interactive **Sales Performance Dashboard** built using Power BI. The dashboard analyzes sales data to provide insights into product performance, regional sales distribution and order trends.

The goal of this project is to transform raw sales data into meaningful visual insights that support better business understanding and decision-making.

## Dashboard Features
- Product-wise sales analysis
- Region-wise sales performance
- Interactive filters and visuals
- Clear and user-friendly dashboard design

## Tools & Technologies
- Power BI
- Data Visualization
- Excel Dataset

##  Insights Generated
The dashboard helps in:
- Identifying top-performing products
- Comparing sales performance across regions
- Understanding sales trends over time
- Analyzing key business metrics
